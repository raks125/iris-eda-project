# Iris EDA Project

This project performs Exploratory Data Analysis (EDA) on the Iris dataset using Python, Pandas, Matplotlib, and Seaborn.

## What this project includes
- Loading the Iris dataset
- Viewing first few records
- Summary statistics
- Species distribution
- Histograms of all features
- Scatter plots
- Insights

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Google Colab
